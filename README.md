This is a basic To-Do List application made using HTML, CSS, and JavaScript.
The app helps users add, display, and manage daily tasks easily.

Users can:

Add a new task using an input field

Mark tasks as completed using a checkbox

Delete tasks when they are finished

JavaScript is used to:

Handle button clicks and checkbox events

Dynamically add and remove tasks from the list

Update the UI without reloading the page

This project helps in understanding:

DOM manipulation

Event handling

Working with input fields and checkboxes

It is a beginner-friendly project and a good example of using JavaScript for real-time interaction
